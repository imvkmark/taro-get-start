# 初步试用 Taro

## 安装和初始化

```
$ pnpm install -g @tarojs/cli

$ taro init myapp

? 请输入项目介绍！ 基础演示站点
? 请选择框架 Vue3
? 是否需要使用 TypeScript ？ Yes
? 请选择 CSS 预处理器（Sass/Less/Stylus） Less
? 请选择模板源 Github（最新）
✔ 拉取远程模板仓库成功！
? 请选择模板 vue3-NutUI（使用 NutUI4.0 的模板）
```

## 使用 yarn

```
$ yarn install
```

## 项目运行

```
$ yarn dev:h5
$ yarn dev:weapp
```
